Olá mundo é uma formoso programa de computador que imprimi "Olá mundo!" (ou "hello word"), usualmente seguindo de uma quebra de linha, com variações como inexistência do ponto de exclamação e letras em maiúscua, no dispositivo de saída. É utilizado como um teste ou como um exemplo de código de uma linguagem
